# NBA 2K26 Zen Scripts - nba2kzenscripts.com

Premium Cronus Zen Scripts for NBA 2K26. Built with React, Vite, and Tailwind CSS.

## Features

- SEO-optimized for keywords: NBA 2K26 Zen Script, NBA Zen Script, 2K26 Zen Script, Cronus Zen Script NBA
- Responsive design
- Product showcase with 3 premium script packages
- Customer testimonials
- Statistics and social proof
- FAQ section with accordion
- Countdown timer for promotional offers

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173) to view the site.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework

## Customization

- Update product information in the `products` array in `App.jsx`
- Modify testimonials in the `testimonials` array
- Adjust pricing and features as needed
- Customize colors in `tailwind.config.js` (primary color is #ff3352)
